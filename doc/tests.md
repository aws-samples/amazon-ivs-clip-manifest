## Packaging commands

**SAM Package**
```sh
sam package \
--template-file template.yaml \
--s3-bucket sam-deploy-clip \
--output-template-file packaged.yaml
```

**SAM Deploy**
```sh
sam deploy \
--template-file packaged.yaml \
--stack-name sample-clip-manifest \
--capabilities CAPABILITY_IAM
```
