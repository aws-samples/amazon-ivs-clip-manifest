# Amazon IVS Clip Manifest - Frontend UI

React application for managing IVS recordings and creating clips with visual controls. Built with [Vite](https://vite.dev/).

## 🔧 Development Setup

**Prerequisites:**
- Node.js 18+ installed

**Local Development (no AWS):**
```bash
npm install
npm run dev:mock    # from project root — starts mock server + UI
```

**Connected to AWS backend:**
```bash
npm run deploy      # from project root — deploys backend, writes config.json
npm run dev         # starts Vite dev server on port 3000
```

## ⚙️ Configuration

The UI resolves API endpoints in this order:

1. `VITE_*` environment variables (highest priority)
2. `src/config.json` (auto-generated by `install.js` from CloudFormation outputs)
3. `src/config.example.json` (committed defaults pointing to mock server)

### Backend → Frontend Config Bridge

When you deploy the backend via `npm run deploy`, the installer:
1. Runs `aws cloudformation describe-stacks --query 'Stacks[].Outputs'`
2. Writes the JSON to `manifest-clip-ui/src/config.json`
3. The Vite app reads it at startup via `import.meta.glob`

The format matches CloudFormation's output structure:
```json
[[
  {"OutputKey": "ApiURLGetRecordings", "OutputValue": "https://xxx.execute-api.region.amazonaws.com/Prod/getrecordings/"},
  {"OutputKey": "ApiURLGetClips", "OutputValue": "https://xxx.execute-api.region.amazonaws.com/Prod/getclips/"},
  {"OutputKey": "ApiURLCreateClip", "OutputValue": "https://xxx.execute-api.region.amazonaws.com/Prod/clipmanifest/"},
  {"OutputKey": "CloudfrontDistribution", "OutputValue": "https://xxx.cloudfront.net"}
]]
```

### Environment Variable Overrides

Copy `.env.example` to `.env.local` and set any overrides:
```bash
VITE_API_RECORDINGS=https://xxx.execute-api.region.amazonaws.com/Prod/getrecordings/
VITE_API_CLIPS=https://xxx.execute-api.region.amazonaws.com/Prod/getclips/
VITE_API_CLIPMANIFEST=https://xxx.execute-api.region.amazonaws.com/Prod/clipmanifest/
```

## 🎨 Components

| Component | Path | Description |
|-----------|------|-------------|
| VideoPlayer | `src/components/player/playerJS.jsx` | Amazon IVS Player + Video.js integration |
| ClipControls | `src/components/ClipControls.jsx` | Timeline with start/end markers |
| ClipsGallery | `src/components/ClipsGallery.jsx` | Grid view of created clips |
| ClipPlayerModal | `src/components/player/ClipPlayerModal.jsx` | Modal for clip playback |
| HomePage | `src/components/HomePage.jsx` | Main layout and workflow |
| IVSChannelInfo | `src/components/IVSChannelInfo.jsx` | Channel config display |
| DebugLog | `src/components/DebugLog.jsx` | API call logging |
